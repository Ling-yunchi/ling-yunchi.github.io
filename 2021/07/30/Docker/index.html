# 镜像的基本命令

  • docker images: 查看所有本地的主机上的镜像

    • -a 列出所有镜像
    • -q 只显示镜像的 id

  • docker search [image name]: 搜索镜像

  • docekr pull: 下载镜像

  • docker image rm: 删除镜像

# 容器的基本命令

  • docker run: 新建容器并启动

  • docker ps: 查询容器

  • 退出容器

  • docker rm: 删除容器
