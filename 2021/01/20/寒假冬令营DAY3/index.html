# 拓展欧几里得

对于不完全为 0 的非负整数 a，b，gcd（a，b）表示 a，b 的最大公约数，必然存在整数对 x，y ，使得 gcd（a，b）=ax+by。

由贝祖定理得：任意两个整数 a,b, 最大公约数为 d=gcd (a,b), 那么对于任意的整数 x,y，ax+by=m, 构成的 m 一定是 d 的整数倍 (即 m% d=0)

证明:

设 a>b。
显然当 b=0，gcd（a，b）=a。此时 x=1，y=0；
当 ab!=0 时
设 ax1+by1=gcd(a,b)ax_1+by_1=gcd(a,b)ax1​+by1​=gcd(a,b);
bx2+(amodb)y2=gcd(b,amodb)bx_2+(a\,mod\,b)y_2=gcd(b,a\,mod\,b)bx2​+(amodb)y2​=gcd(b,amodb);
根据朴素的欧几里德原理有 gcd(a,b)=gcd(b,amodb)gcd(a,b)=gcd(b,a \,mod\,b)gcd(a,b)=gcd(b,amodb);
则:ax1+by1=bx2+(amodb)y2ax_1+by_1=bx_2+(a \,mod \,b)y_2ax1​+by1​=bx2​+(amodb)y2​;
即:ax1+by1=bx2+(a−⌊ab⌋∗b)y2=ay2+bx2−⌊ab⌋∗by2ax_1+by_1=bx_2+(a-\left \lfloor\frac{a}{b}\right \rfloor*b)y_2=ay_2+bx_2-\left \lfloor\frac{a}{b}\right \rfloor*by_2ax1​+by1​=bx2​+(a−⌊ba​⌋∗b)y2​=ay2​+bx2​−⌊ba​⌋∗by2​;

ax + by = gcd (a,b) 是一个不定方程，有多解是一定的，但是只要我们找到一组特殊的解 x0 和 y0 那么，我们就可以用 x0 和 y0 表示出整个不定方程的通解，因为，将 (x0,y0) 代入 ax+by=c，则有

a∗x0+b∗y0=ca*x_0+b*y_0=ca∗x0​+b∗y0​=c

通过拆添项，可有：

a∗(x0+b)+b∗(y0−a)=ca*(x_0+b)+b*(y_0-a)=ca∗(x0​+b)+b∗(y0​−a)=c

a∗(x0+2∗b)+b∗(y0−2∗a)=ca*(x_0+2*b)+b*(y_0-2*a)=ca∗(x0​+2∗b)+b∗(y0​−2∗a)=c

a∗(x0+3∗b)+b∗(y0−3∗a)=ca*(x_0+3*b)+b*(y_0-3*a)=ca∗(x0​+3∗b)+b∗(y0​−3∗a)=c

……

a∗(x0+k∗b)+b∗(y0−k∗a)=c(k∈Z)a*(x_0+k*b)+b*(y_0-k*a)=c (k∈Z)a∗(x0​+k∗b)+b∗(y0​−k∗a)=c(k∈Z)

又因为我们已知:

$aX + bY = gcd(a,b) $

两边同时乘上\frac{c}

得到:a∗cXgcd(a,b)+b∗cYgcd(a,b)=ca*\frac{cX}{gcd(a,b)}+b*\frac{cY}{gcd(a,b)}=ca∗gcd(a,b)cX​+b∗gcd(a,b)cY​=c

x=x0+bgcd(a,b)∗kx = x_0 + \frac{b}{gcd(a,b)}*kx=x0​+gcd(a,b)b​∗k

这样，所有满足 ax+by=c 的可行解都可求出。

那我们怎么求出一对特解 x、y 呢？

由上面的证明可知，对于欧几里得算法，当 a 或 b 有一个为零时，令一个数即为所求的 gcd。所以，欧几里得算法的停止状态为：a=gcd，b=0。

此时，我们为了满足 ax + by = gcd ，可以得出 x=1，y=0。

那么最终状态的值我们就都知道了，x=1、y=0、a=gcd、b=0，而递推式也已经给出，那我们完全可以在求 gcd 的同时，在最终状态对 x，y 进行赋值，然后再在递归回溯的时候依次更新 x，y 的值，返回 gcd 的时候，生成的 x，y 就是一对特解。

特殊形式a∗x+b∗y=gcd(a,b)a*x+b*y=gcd(a,b)a∗x+b∗y=gcd(a,b) 代码实现:

int exgcd(int a, int b, int& x, int& y)
{
    if (b == 0)
    {
        x = 1; y = 0;
        return a;
    }
    int t = exgcd(b, a % b, x, y);
    int x_0 = x, y_0 = y;
    x = y_0, y = x_0 * (a / b) * y_0;
    return t;
}
