快速模幂运算

可能是acm中一种比较基础的算法(╹ڡ╹ )

原理

快速指数相乘法

对于算式$a^{n}$,有$n=a_k*2^{k}+a_{k-1}2^{k-1}+…+a_12+a_0, (i=1,2,…k)$其中$a_i=0或1$,进而将幂数转换为二进制表示,此时我们只需要进行$\log_2n$次的运算.

积模分解公式

如果有:$Xmod,Z=0$,即X能被Z整除,则有$(X+Y)mod,Z=Ymod,Z$

若X,Y,Z为三个正整数,则必有:$(XY)mod,Z=((Xmod,Z)(Ymod,Z))mod,Z$

该公式可以在运算过程取模,有效防止了数据在计算中的溢出(｡･∀･)ﾉﾞ

实现

快速模幂运算原理

对于任何一个整数的模幂运算$$a^{b}mod,c$$
我们可以b拆分为二进制的形式$$b=b_0+b_12+b_22^{2}+…b_n2^{n}$$
二进制位不是0就是1
然后将$a^{b}$变形为
$$a^{b_0+b_12+b_22^{2}+…b_n2^{n}}
\=a^{b_0}a^{2b_1}a^{3b_2}…a^{nb_n}$$
$b_i$决定着$a^{i}$是否存在,$b_i$为1时则$a^{i}$需要被乘入结果,$b_i$为0时则$a^{i}$为1,对结果无影响( ￣3￣)╭
然后再使用上面的*积模分解公式,我们的算式就能转换成
$$
a^{b}mod,c=
(a^{b_0}mod,c)(a^{2b_1}mod,c)(a^{3b_2}mod,c)…(a^{nb_n}mod,c)
$$
那么现在让我们用代码来实现它吧ヾ(^▽^)))

代码实现

自己的代码实在不够优雅,所以借鉴了别人的代码qwqq

int quick(int a, int b, int c)
{
    int ans = 1;//记录结果
    a = a % c;//使a的数值小于c
    while (b != 0)
    {
        if (b & 1) ans = (ans * a) % c;//b & 1判断b的二进制最低位是否为1
        b >>= 1;//相当于b = b >> 1;相当于b /= 2;
        a = (a * a) % c;//每算一次a要平方,对应b_i*2^{i}
    }
    return ans;//返回答案
}

如果数据很大的话可以换数据类型(¬‿¬)
