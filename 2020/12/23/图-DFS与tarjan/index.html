DFS与tarjan

DFS

DFS的全称为depth-first search,深度优先搜索,顾名思义,就是优先访问”深”节点的遍历方法,它具有一下几条访问规则

  1. 假设初始状态所有顶点都没有被访问,然后从某一顶点v出发,访问先访问顶点v
  2. 然后选择v的一个邻接点出发,访问该邻接点
  3. 若该邻接点有邻接点时,继续访问
  4. 直到所访问的点没有邻接点为止,开始回溯
  5. 若回溯到的点存在未被访问的邻接点时,从该邻接点处继续按上面规则访问
  6. 直到回溯至顶点v
  7. 若图中还有未被访问的点,则选择一个未被访问的点,重复上面操作直到整个图被访问完为止

所以DFS的每一次搜索都是进入的最底端,无法继续访问之后才会回头(不撞南墙不回头＞︿＜

DFS遍历方式

方式一:先访问当前节点再递归相邻节点

//DFS实现
#include<stdio.h>
#include<stdlib.h>
#define MAXVEX 100
int read[MAXVEX] = { 0 };

typedef struct EdgeNode //边表结点
{
    int adjvex;//邻接点域,存储该顶点对应的下标
    //int weight;
    struct EdgeNode* next;//链域,存储下一个顶点
}EdgeNode;

typedef struct VertexNode //顶点表结点
{
    int data;
    EdgeNode* firstedge;//边表头指针
}VertexNode, AdjList[MAXVEX];

typedef struct
{
    AdjList adjList;
    int numVertexes, numEdges;//顶点数与边数
}GraphAdjList;

void CreatALGraph(GraphAdjList* G)
{
    EdgeNode* e;
    printf("enter vertex and edge:\n");
    scanf("%d%d", &G->numVertexes, &G->numEdges);
    //建立定点表
    for (int i = 0; i < G->numVertexes; i++)
    {
        scanf("%d", &G->adjList[i].data);//读入顶点信息
        G->adjList[i].firstedge = NULL;//将边表置为空表
    }
    //建立边表,有向图
    for (int k = 0; k < G->numEdges; k++)
    {
        int i, j;
        printf("enter vertexnum on vi vj:\n");
        scanf("%d%d", &i, &j);
        //头插法
        e = (EdgeNode*)malloc(sizeof(EdgeNode));

        e->adjvex = j;//邻接序号为j
        e->next = G->adjList[i].firstedge;//将e指针指向当前顶点指向的节点
        G->adjList[i].firstedge = e;//将当前顶点的指针指向e
        //表示i可以到达j
    }
}

typedef int Boolean;
Boolean visited[MAXVEX] = { 0 };//标记访问的数组
//深度优先递归算法
void DFS(GraphAdjList* GL, int i)
{
    EdgeNode* p;
    visited[i] = 1;
    printf("%d ", GL->adjList[i].data);//打印顶点
    p = GL->adjList[i].firstedge;
    while (p != NULL )
    {
        if (!visited[p->adjvex])
            DFS(GL, p->adjvex);
        p = p->next;
    }
}
//邻接表的深度遍历操作
void DFSTraverse(GraphAdjList* GL)
{
    for (int i = 0; i < GL->numVertexes; i++)
        visited[i] = 0;
    for (int i = 0; i < GL->numVertexes; i++)
        if (!visited[i])
            DFS(GL, i);
}

int main()
{
    GraphAdjList map;
    CreatALGraph(&map);
    DFSTraverse(&map);
    //while (1);
    
    return 0;
}

tarjan

强连通分量

连通 :在无向图中,从任意点i出发可以到达任意点j(可以不止经过一条边)

强连通 :有向图中,从任意点i出发可以到达任意点j

弱连通 :把有向图看成无向图时,从任意点i出发可以到达任意点j

强连通分量 :有向图中,存在某一些区域使得该区域中的点满足从任意点i出发可以到达任意点j,则称该块区域为整个图的一个强连通分量.

tarjan求图的强连通分量

对于每一个顶点$x$,我们赋予它两个属性$i,j$
$i$ :DFS中x点被访问的时间点
$j$ :x通过有向边,可回溯到的最早的时间点
$$x[i][j]$$
我们可以建立两个数组来记录每个顶点的这两个属性
$$dfn[]$$
就是一个时间戳（被搜到的次序），一旦某个点被DFS到后，这个时间戳就不再改变（且每个点只有唯一的时间戳）。所以常根据dfn的值来判断是否需要进行进一步的深搜。
$$low[]$$
该子树中，且仍在栈中的最小时间戳，像是确立了一个关系，low[ ]相等的点在同一强连通分量中。

注意初始化时 dfn[ ] = low[ ] = ++cnt

思路

选定一个入点,将其入栈,同时对其时间戳进行初始化

cnt=1;
dfn[ ] = low[ ] = ++cnt

然后不停将访问到的点入栈并递归初始化
因为我们使用的是第二中DFS遍历方式
在递归到最底层后若没有可以继续DFS时则开始回溯更新low[]
$$…j→k→i　　　i→j$$
若该点i可以连接到一个时间点比他早的点j,则更新low[j]的数值为dfn[i]
回溯一次后点k的low[k]若小于点i的low[i],则将low[k]的数值更新为low[i]
回溯到j点时,因为j没有连通到比他更早的点,所以他的dfn[j]=low[j],此时执行出栈操作直到点j出栈
则i j k为一个强连通分量
